# 身份模块 (Identity Module)

您的数字身份基石。此模块定义了您是谁、您如何交流以及您代表什么。

## 模块内的文件

| 文件 | 用途 |
|------|---------|
| `voice.md` | 语气、风格、写作模式 |
| `brand.md` | 定位、话题、受众 |
| `values.yaml` | 核心信念和原则 |
| `bio-variants.md` | 适用于不同平台的各种长度的个人简介 |
| `prompts/` | 用于内容生成的各种可重用提示词 |

## 何时使用

- **撰写任何内容时**：请先阅读 `voice.md`。
- **创建新平台资料时**：查看 `bio-variants.md`。
- **做出战略决策时**：参考 `values.yaml`。
- **确定内容话题时**：咨询 `brand.md` 以了解定位。

## 智能体指令

<instructions>
为用户创作内容时：
1. 在起草之前，**务必**阅读 voice.md。
2. 匹配其能量水平、词汇和结构模式。
3. 避免使用 “禁止使用” 部分列出的词汇/短语。
4. 自然地融入其标志性短语。
5. 检查 brand.md 以确保话题相关性。
</instructions>

## 声音快速参考

有关详细的声音指南，请参阅 [voice.md](voice.md)。

智能体应内化的关键要素：
- 沟通风格（正式/随意的程度）
- 标志性短语和词汇
- 结构模式（帖子格式、钩子）
- 要强调的话题与要避免的话题
