# LinkedIn 帖子模板

## 元数据
```yaml
话题: "[占位符]"
内容支柱: "[占位符：内容支柱名称]"
格式: "story (故事)|lesson (教训)|hot_take (鲜明观点)|how_to (指南)|list (列表)"
```

---

## 钩子 (首 2-3 行)
<!-- 必须在点击“查看更多”之前就能吸引人 -->
```
[占位符：引导读者点击“查看更多”的强力开场]
```

---

## 正文

### 格式：故事
```
背景设定：
[占位符：情境/背景]

挑战：
[占位符：发生了什么/面临的问题]

转折点：
[占位符：发生了什么变化]

教训：
[占位符：学到了什么]
```

### 格式：教训/指南
```
以下是如何 [占位符：取得某种结果]：

1. [步骤/要点]
[占位符：简要解释]

2. [步骤/要点]
[占位符：简要解释]

3. [步骤/要点]
[占位符：简要解释]

核心见解：
[占位符]
```

### 格式：鲜明观点
```
[占位符：具有争议性的陈述]

原因如下：

[占位符：支持论据 1]

[占位符：支持论据 2]

[占位符：补充细节或前提条件]
```

---

## 结语
```
[占位符：总结或引导讨论]
```

---

## 互动钩子
<!-- 鼓励评论 -->
```
[占位符：向受众提出的问题]

示例：
- “你对此怎么看？”
- “你经历过这种情况吗？”
- “你还有什么想要补充的吗？”
```

---

## 标签 (最多 3-5 个)
```
#[占位符] #[占位符] #[占位符]
```

---

## 发布前检查清单
- [ ] 钩子在首 2-3 行内是否起作用？
- [ ] 是否使用了换行符以提高可读性？
- [ ] 价值点是否既显专业又不失人性化？
- [ ] 是否以互动提示结束？
- [ ] 语气是否匹配品牌形象？（LinkedIn 版通常稍微正式一点）
- [ ] 是否避免了过度自我推销？
